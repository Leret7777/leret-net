What to include in an enquiry

A booking enquiry gets a faster answer when it covers:

  • Brand / client and a link to your work
  • Shoot date(s) and location
  • Type of work (editorial, e-commerce, campaign, runway…)
  • Usage — where and how long the images will run
  • Rate or budget range
